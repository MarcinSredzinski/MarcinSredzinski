  • About me

    I'm Marcin. I code in .NET, but you already know that. What you don't know, it's that I'm not some stereotypical hardcore programmer hidden behind a keyboard, unresponsive until a task is completed and only leaving the basement to get some coffee.
    I'm a team player.
    That's why when we work together I need to understand your goal, see the big picture. Only then I can truly help you. And the code? The code is a tool to help with this goal.

How can I help you:

Discovery

In this dynamic world, being able to quickly verify the ideas is a must. Whether you want to create a new project from scratch, or migrate existing project into a new tech stack, I can help you with:

  • designing architecture,
  • prototyping,
  • setting up CI/CD
  • setting up monitoring
  • geting the usage statistics,
  • running experiments.
